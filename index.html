The main aim of the Data For Climate Action Edinburgh (D4CAE) group is to help other climate volunteers around Edinburgh, using our data skills.

We have a GitHub repository that we use to make a large proportion of our data, our code and so on publicly available, under an open licence.

If you would like help with climate or biodiversity data, please get in touch - data4climateactionedinburgh@gmail.com !
